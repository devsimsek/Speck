# Speck

Speck is a web-based sensor analysis and dust mapping utility for photographers.

It helps you visualize sensor dust, map problem spots, inspect camera metadata, and generate a reusable reference map for cleaning and maintenance.

## What Speck does

- Load local JPEG and PNG images directly in the browser
- Read EXIF data, including shutter count and lens information when available
- Replicate Lightroom-style visual spot inspection with high-contrast filters
- Overlay a grid to scan images methodically
- Drop and export dust markers on top of the original image
- Use a full-screen light panel for camera and lens inspection
- Work offline as a PWA after the first load

## Quick start

1. Open Speck in a modern browser.
2. Load a blank sky, white wall, or evenly lit image.
3. Enable **Visualize Spots** to reveal dust more clearly.
4. Use **Dust Mapping Mode** to place markers.
5. Export the map as a JPEG when finished.

## Keyboard shortcuts

### Light Screen Mode
- `Esc` — Exit
- `W` — Pure White
- `B` — Pure Black
- `R` — Pure Red
- `G` — Pure Green
- `Shift + B` — Pure Blue

### General
- `Ctrl/Cmd + U` — Check for updates
